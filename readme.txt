This plugin shows all the related posts according to the categories of the displayed post.

== Description ==
This plugin shows the related posts under a post content according to the categories assigned to the current viewed post.
This is a very light plugin which loads very fast and compatible with all themes.

<h2>Features</h2>
1. Lightweight plugin
2. Shows posts according to the categories.
3. Compatible with all theme designs.
4. Can control the number of posts to be displayed.


== Installation ==

The installation of this plugin is very simple. Just follow these simple steps

e.g.

1. Upload `perfect-related-posts` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Change the settings from the 'Perfect Related Posts' under the admin dashboard.

== Frequently Asked Questions ==

= How many related posts can be shown ? =

Any number of related posts can be shown depending upon the posts that viewed post categories has.
